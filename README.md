# AI Agent Ecosystem Visualizer

> 互動式 AI Agent 生態系統力引導圖 — 直觀展示各大 AI Agent 框架、模型、平台之間的關係網絡

## 功能特色
- 🌐 **力引導圖** — 自動佈局的網路拓撲圖
- 🏷️ **分類篩選** — 按 Framework/Model/Desktop/Platform/Tool 篩選
- 💡 **互動操作** — 拖曳節點、縮放、懸停提示、點擊詳情
- 📊 **即時統計** — 節點/連線數 + FPS 顯示
- ⚡ **物理引擎** — 可開關的力引導模擬
- 📱 **觸控支援** — 手機平板也能操作

## 快速開始

直接用瀏覽器開啟 `index.html` 即可開始探索。

### 操作說明
- **拖曳節點** — 點擊並拖移任意 Agent
- **縮放** — 滑鼠滾輪 / 觸控縮放
- **平移** — 拖曳空白區域
- **篩選** — 左側分類按鈕切換顯示
- **懸停** — 查看 Agent 詳細資訊
- **點擊** — 顯示 Agent 詳情面板
